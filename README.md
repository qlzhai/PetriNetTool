# PetriNetTool
一种用于Petri网模型分析的软件工具，使用C++编写，分别在Windows和Linux下完成开发，其中，Windows在Visual Studio中使用Qt库开发，Linux下使用VS Code开发

分支说明：

Windows分支下：

      存放的是Windows操作系统下的源代码文件
      
      所使用的软件为Visual Studio 2019、 Qt6.6.0
      
Linux分支下：
      存放的是Linux操作系统的源代码文件
      
      所使用的软件为VS code、Ubuntu 18.04，在wsl子系统下完成开发
